# Экспорт текстур из Photoshop

### В работе с Unreal Engine лучше использовать формат текстур .TGA 32 bit.

Формат TGA предпочтительнее PNG в Unreal Engine по нескольким причинам, связанным с особенностями графических данных и их использованием в игровом движке:

1. Более простая компрессия: TGA — это некомпрессированный формат, что делает его чтение и обработку быстрее. PNG использует сжатие с потерями, что требует дополнительных ресурсов для декомпрессии при загрузке в память, особенно при загрузке большого количества текстур.
2. Поддержка альфа-канала без артефактов: Хотя PNG также поддерживает альфа-канал (прозрачность), TGA лучше сохраняет качество альфа-канала без артефактов, особенно при работе с масками прозрачности.
3. Размеры файлов: TGA может быть больше по размеру, но при этом этот формат лучше оптимизирован для быстрой загрузки и рендеринга внутри движка.
4. Поддержка 32-битных текстур: TGA поддерживает текстуры с более высокой точностью цветов, что делает его подходящим для использования в материалах с высокой детализацией.
5. Совместимость с движком: Unreal Engine исторически поддерживает TGA и часто использует этот формат по умолчанию для работы с текстурами, особенно в разработке игр для консолей или платформ, где важна производительность.

Таким образом, формат TGA более предпочтителен, чем PNG.
